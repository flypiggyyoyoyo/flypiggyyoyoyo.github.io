学习SQL写法的笔记

JOIN

数据分散到多个表时，需要用表连接来关联数据

结构

内连接

只返回两表之中满足条件的记录（交集）

左连接

返回左表（表 A）的 所有记录，右表（表 B）中匹配的记录，若右表无匹配则用 NULL 填充

右链接

返回右表（表 B）的 所有记录，左表（表 A）中匹配的记录，若左表无匹配则用 NULL 填充

全外链接

返回两表的 所有记录，匹配的记录合并，不匹配的字段用 NULL 填充（并集）
